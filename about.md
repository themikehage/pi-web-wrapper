# pi-web-wrapper
**Type:** PRODUCTION
**Description:** Web interface for pi coding agent with real-time streaming, multi-session chat, and user authentication. Wraps the @earendil-works/pi-coding-agent SDK.
**Stack:** Bun, Hono, React, Vite, TypeScript, Tailwind CSS v4, Framer Motion, WebSocket
**Deployment Target:** Coolify
**Database Tier:** No database (localStorage client-side, filesystem sessions server-side)
